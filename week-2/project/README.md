# WEB103 Project 2 - Codelock

Submitted by: Mubarak Odufade

About this web app: This is a full stack application for a workshop series called Codelock. The workshop series include leetcode challenges and solution videos, and the website presents details about each workshop to users. Users can find the related conceps and algorithms covered in each workshop, the solution explanation video, the leetcode/problem links, difficulty, rating, and a link to the discord server.

Time spent:4 hours
